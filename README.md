# The Glossary Support

Minimal public support site for The Glossary iOS app.

To use this for App Store Connect:

1. Replace the placeholder contact details in `index.html`.
2. In GitHub, open this repo's `Settings` -> `Pages`.
3. Set the publishing source to the `main` branch and the root folder.

The published support URL will be:

`https://jaredflomen.github.io/the-glossary-support/`
